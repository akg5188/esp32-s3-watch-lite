# Watch Lite 使用说明

## 按键

- 短按 BOOT：从深睡唤醒；亮屏时回到首页并刷新行情。
- 长按 BOOT 约 2 秒：打开配置热点，方便手机进入设置页。

## 首页

首页显示行情列表、Wi-Fi 状态、时间、电量信息。手表联网后会刷新行情；如果短时间内网络不通，会先显示上次缓存。

目前行情来源包括 CoinGecko、Stooq 和 Yahoo Chart。周末或海外源不可用时，部分股票/指数可能停留在上一次交易日数据，这是正常现象。

## AI 页面

进入 AI 页面后可以用语音或文本触发问答。API URL、API Key、模型和语音网关地址都在配置网页里设置。

如果使用 OpenAI 兼容中转站，常见填写方式是：

```text
API URL: https://example.com/v1
API Key: sk-...
Model: 由中转站支持的模型名
```

如果直接使用豆包/火山方舟文本聊天，可以在配置页填写：

```text
API URL: https://ark.cn-beijing.volces.com/api/v3
API Key: 火山方舟 API Key
Model: 方舟控制台里启用的豆包模型或 endpoint id
```

首页左右两侧有多个 AI 快捷入口：`GPT`、`Gem`、`豆包`、`DS`、`千问`、`自定`。配置网页里每个 AI 都可以保存独立的 API URL、API Key 和 Model；点击首页入口会切换到对应 AI 后进入聊天页。

语音链路可以直接走中转站，也可以用 `tools/voice_gateway.py` 或 Docker 语音网关做 ASR/TTS/联网资料整理。

## 配置热点

默认省电策略下，手表已经有 Wi-Fi 配置时不会常驻 AP。需要修改配置时：

1. 长按 BOOT 约 2 秒。
2. 手机连接 `WatchLite-xxxx`。
3. 密码 `12345678`。
4. 浏览器打开 `http://192.168.4.1`。
5. 保存配置后手表会继续按新配置联网。

## 多 Wi-Fi

设置页支持最多 5 个 Wi-Fi。每个 Wi-Fi 可以保存独立的：

- SSID / 密码
- 静态 IP
- 子网掩码
- 网关
- DNS1 / DNS2

点选不同 Wi-Fi 后，手表会按该 Wi-Fi 对应的网络参数连接。

## 省电行为

- 蓝牙关闭。
- 配置 AP 平时关闭，只有长按 BOOT 或进入 Wi-Fi 扫描/切换时才开启。
- Wi-Fi 用完会断开，息屏后保持 STA 省电模式。
- 音频功放默认关闭，录音/播放结束后关闭。
- 摇晃唤醒 IMU 默认不启动。
- 息屏后约 15 秒进入 deep sleep。
- deep sleep 后只能通过 BOOT 按键唤醒。

亮屏等待时间和亮度当前不继续压缩，避免影响正常使用体验。

## 常见问题

如果开机后看不到配置热点，通常是因为手表已经保存了 Wi-Fi，进入省电模式不会常驻 AP。长按 BOOT 约 2 秒即可打开。

如果行情时间不更新，先确认 Wi-Fi 是否连接成功；周末或行情源不可用时，部分品种可能不会变。

如果 AI 回复慢，优先检查中转站模型速度、ASR/TTS 网关速度和网络延迟。
